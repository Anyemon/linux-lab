# Práctica 00 - Creación del entorno de laboratorio

##Objetivo
Preparar un entorno de laboratorio para aprender a administración de sistemas Linux y fundamentos de ciberseguridad.
- Sistema operativo: Debian 12
- Entorno de escritorio: XFCE
- Tipo de instalación: Hardware físico
- CPU: 4 núcleos
- Memoria RAM: 4 GB
- Disco: 200 G

Se decidió instalar Debian directamente en hardware físico para obtener experiencia real de administración de sistemas, evitanod las limitaciones de un entorno virtualizaddo.

## Plan de expansión el laboratorio

Como parte de la evolución del laboratorio, se prevé implementar un entorno virtualizado dentro ded Debian para realizar prácticas avanzadas.

# Objetivos:
- Crear múltiples máquinas virtuales aisladas.
- Simular redes internas.

- Realizar pruebas dde conectividad y segmentación de red.
